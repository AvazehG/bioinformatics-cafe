YAML to JSON converter
======================

Converter from YAML to JSON format. See `yaml2json.py -h` for help.

Usage
-----

```
yaml2json.py [stdin|file] > out.json 
```

Requirments
-----------

* python 2 or 3 with package [PyYAML](http://pyyaml.org/wiki/PyYAML) installed
